A "proxy" that lets you bypass website blocks from Lightspeed Filter.

## How does Unspeed work exactly?
I have to be vague in order for the patch to take longer, but while decompiling, I've noticed Lightspeed Filter has a flaw where if a certain URL contains a specific string, the request will not be intercepted. I've tested it with a bookmark and it indeed worked! The tab got blocked after like half a second. 

A few days later, I had the thought of embedding a URL inside an iframe with this bug and it allowed me to visit a stripped down version of a blocked website!
